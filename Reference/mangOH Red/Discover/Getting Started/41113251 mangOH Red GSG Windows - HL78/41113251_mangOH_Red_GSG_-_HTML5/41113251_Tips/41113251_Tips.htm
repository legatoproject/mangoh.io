6: Tips

6.1 Finding your SIM’s APN

Your SIM provider should give you the APN that you use to connect to their network. If you don’t have the APN, you should be able to find it online.

If your mobile network operator uses different APNs for 3G and LTE, make sure to use the APN for the correct network based on your CF3 module type (e.g. WP8548 is a 3G-only module — the APN for the network operator’s 3G network should be used).

Examples:

·Sierra Wireless — internet.sierrawireless.com

· Rogers Wireless — internet.com (GPRS); ltemobile.apn (LTE)

·Others — Search the Internet for "<provider> APN". For example, "Rogers Wireless APN"
